# 📊 Marketing Data Pipeline

## 📌 Visão Geral
Este projeto tem como objetivo construir um **pipeline de dados para análise de campanhas de marketing** utilizando SQL. Ele cobre todo o processo, desde dados brutos até a criação de estruturas prontas para ferramentas de Business Intelligence.

---

## 🛠️ Tecnologias Utilizadas
- SQL (modelagem de dados, transformações e views)
- Power BI *(em desenvolvimento)*
- Excel (fonte de dados)

---

## ⚙️ Estrutura do Projeto

### 1. Fonte de Dados
- Dataset de marketing em formato CSV

### 2. Limpeza de Dados
- Tratamento de valores nulos  
- Padronização de colunas  
- Preparação dos dados para análise  

### 3. Modelagem de Dados
Criação de tabelas e views estruturadas:
- `vw_customer_profile`
- `vw_customer_activity`
- `vw_customer_spending`
- `vw_campaign_performance`

### 4. Regras de Negócio
- Análise de comportamento do cliente  
- Padrões de compra  
- Performance de campanhas  

### 5. Dashboard *(Em desenvolvimento)*
- KPIs (clientes, compras, gastos)
- Análise de comportamento
- Performance de campanhas
- Análise por categoria de produto

---

## 📈 Objetivos
- Entender o comportamento dos clientes  
- Identificar campanhas mais eficientes  
- Analisar padrões de consumo  
- Criar uma base organizada para BI  

---

- Criação de visualizações e insights
